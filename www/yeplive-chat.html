Yeplive Chat Server

Task

Built a custom chat server for all Yeplive clients to handle thousands of concurrent users. Ensuring scalability and reliabilty is a must. The service must have well written documentation.

Summary

Custom built chat server for all Yeplive clients to handle thousands

Created extensive documentation for the Websocket API to be consumed by all clients

Communicated with existing backend API for persistance of data

Used best practices to ensure error free delivery of messages

Elastic Beanstalk ensured error free deployments

Node.js and Socket.io ensured reliatvility and scalabilty with sticky sessions and redis based caching served via Elasticache.
